## Описание приложения

**Zapret Desktop Linux** — это форк версия оригинального проекта [ZapretDYLinux](https://github.com/Sergeydigl3/zapret-discord-youtube-linux), предназначенная для удобного доступа к различным сервисам. Наша программа предлагает интуитивно понятный и красивый интерфейс, который делает использование приложения простым и приятным.

## Текущий статус

На данный момент приложение находится на этапе **beta**. Мы активно работаем над улучшениями и обновлениями, чтобы сделать его еще лучше для наших пользователей.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/peshk0v/ZapretDesktop.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd ZapretDesktop
   ```

3. Запустите приложение:
   ```bash
   python app.py
   ```

---

## Использование

После запуска приложения вы сможете выбрать различные сервисы в красивом интерфейсе. Пожалуйста, примите во внимание, что приложение находится в стадии beta, и некоторые функции могут быть нестабильны.
